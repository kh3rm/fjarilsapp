# Annies fjärilar

Fjärilapp med två vyer: fjärilsarter och egen samling.

## Syfte 

Att kunna användas till att identifiera och lära sig om fjärilar, fotografera dem och spara till sin egen samling.


## System

- Statisk webbapp för GitHub Pages
- HTML, CSS och JavaScript
- Supabase för inloggning, databas och bildlagring
- Lokala referensbilder i WebP-format
- PWA med manifest och service worker
- Ingen serverkod
